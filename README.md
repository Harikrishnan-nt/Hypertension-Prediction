Hypertension Risk Prediction using Machine Learning

Problem Statement

Hypertension is a major risk factor for cardiovascular disease.
This project aims to build a machine learning model that predicts whether an individual is at risk of hypertension using clinical and demographic features.

⸻

Dataset
 • Source: Kaggle
 • Number of records: 1913
 • Features: Age, Gender, Resting Blood Pressure, Cholesterol, BMI, and related clinical attributes
 • Target variable: Hypertension (Yes / No)

⸻

Approach
 1. Data cleaning and preprocessing
 2. Exploratory Data Analysis (EDA)
 3. Feature engineering and encoding
 4. Model training and comparison
 5. Model evaluation

⸻

Models Used
 • Logistic Regression
 • Decision Tree
 • Random Forest
 • XG Boost



⸻

Evaluation Metrics
 • Accuracy
 • Precision
 • Recall
 • ROC-AUC

⸻

Results

The trained models were evaluated using standard classification metrics.
Feature engineering and model selection improved predictive performance over baseline models.
